# Jeskars 

Jeskars is a modern car marketplace website designed to showcase premium vehicles, services, and customer experiences in a clean and user-friendly interface.

---

##  Features

- Responsive homepage with hero section
- Car store page with featured vehicles
- Services page
- Customer stories page
- Contact form for user inquiries
- Consistent layout using Tailwind CSS

---

##  Pages

- `index.html` → Homepage  
- `store.html` → Shop cars  
- `services.html` → Services offered  
- `stories.html` → Customer stories  
- `about.html` → About the company  
- `contact.html` → Contact form  

---

- HTML
- Tailwind CSS
